# Advanced SQL Puzzles

Working through SQL puzzles to solidify my SQL skills.

## Source

Puzzles from [Advanced SQL Puzzles](https://github.com/smpetersgithub/AdvancedSQLPuzzles) by Scott Peters.

## Structure

```
├── problems.md              # Problem descriptions
├── ddl.sql                  # Original T-SQL table definitions
├── solutions.sql            # Original T-SQL solutions
├── ddl_postgresql.sql       # PostgreSQL table definitions
├── solutions_postgresql.sql # PostgreSQL solutions
└── my_solutions/            # My attempts at solving the puzzles
```
